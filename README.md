# Image Processor

### You can add/upload images to it and edit them as you wish

## Features it provides
- Cropping image
- Rotate an image
- Resize image
- Flipping an image
- Mirroring an image
- Adding filters : Sepia, Grayscale, Invert

## To clone this repo simply:
```
git clone "git@github.com:Risbern21/Image-Processor.git"
```
